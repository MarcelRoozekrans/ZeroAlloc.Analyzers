# ZA0107: Pre-size collections when the capacity is known

## Description

Detects `new List<T>()` with no capacity argument when `.AddRange()` is called on the variable in the same method. Creating a list without a capacity hint and then bulk-adding items causes repeated internal array reallocations. Pass the known size to the constructor instead.

## Violation

```csharp
var list = new List<int>();
list.AddRange(source);
```

## Fix

```csharp
var list = new List<int>(source.Length);
list.AddRange(source);
```

## When to suppress

- The source size is truly unknown or highly variable and pre-sizing would waste memory

## Minimum TFM

Any
