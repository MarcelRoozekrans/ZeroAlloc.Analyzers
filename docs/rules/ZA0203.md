# ZA0203: Use AsSpan instead of Substring

## Description

Detects `string.Substring()` calls that allocate a new string. Use `AsSpan()` or `AsSpan(start, length)` to avoid the allocation when possible.

## Violation

```csharp
var sub = text.Substring(5);
var mid = text.Substring(2, 4);
```

## Fix

```csharp
var sub = text.AsSpan(5);
var mid = text.AsSpan(2, 4);
```

## When to suppress

- The result must be a `string` (e.g., passed to an API that doesn't accept `ReadOnlySpan<char>`)

## Minimum TFM

net5.0
