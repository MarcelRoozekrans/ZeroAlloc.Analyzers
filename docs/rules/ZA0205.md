# ZA0205: Use CompositeFormat.Parse to cache format strings

## Description

Detects `string.Format(...)` calls where the format string is a constant or static readonly field. In these cases, `CompositeFormat.Parse()` can be used to parse the format string once and cache it, avoiding repeated parsing on each call.

## Violation

```csharp
private static readonly string Fmt = "Hello {0}, you have {1} items";

var message = string.Format(Fmt, name, count);
```

## Fix

```csharp
private static readonly CompositeFormat Fmt =
    CompositeFormat.Parse("Hello {0}, you have {1} items");

var message = string.Format(null, Fmt, name, count);
```

## When to suppress

- The format call is not on a hot path
- The format string is used only once

## Minimum TFM

net8.0
