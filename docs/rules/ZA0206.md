# ZA0206: Avoid converting Span to string before Parse

## Description

Detects patterns like `int.Parse(span.ToString())` where a `ReadOnlySpan<char>` or `Span<char>` is converted to a string just to be passed to a `Parse` method. Since .NET 6, most numeric types and common types like `DateTime`, `Guid`, etc. provide `Parse(ReadOnlySpan<char>)` overloads that avoid the intermediate string allocation.

## Violation

```csharp
ReadOnlySpan<char> span = "42".AsSpan();
var n = int.Parse(span.ToString());
```

## Fix

```csharp
ReadOnlySpan<char> span = "42".AsSpan();
var n = int.Parse(span);
```

## When to suppress

- The allocation is not on a hot path

## Minimum TFM

net6.0
