# ZA0602: Avoid params calls in loops

## Description

Detects method calls with `params` arrays inside loops. The compiler allocates a new array for each call when individual arguments are passed to a `params` parameter.

## Violation

```csharp
for (int i = 0; i < 100; i++)
{
    Log("value: {0}", i);  // allocates object[] on every iteration
}
```

## Fix

```csharp
// Use a non-params overload or pre-allocate the array
var args = new object[1];
for (int i = 0; i < 100; i++)
{
    args[0] = i;
    Log("value: {0}", args);
}
```

## When to suppress

- The loop has few iterations and is not in a hot path

## Minimum TFM

Any
