# ZA0802: Avoid Enum.ToString() — allocates a string

## Description

Detects `.ToString()` calls on enum values. Each call allocates a new string via reflection. Consider using a cached dictionary, `nameof()`, or a switch expression instead.

## Violation

```csharp
var name = myEnum.ToString();
```

## Fix

```csharp
// Option 1: nameof for known values
var name = nameof(MyEnum.Value);

// Option 2: cached lookup
private static readonly Dictionary<MyEnum, string> Names = new()
{
    [MyEnum.Value] = nameof(MyEnum.Value),
};
var name = Names[myEnum];
```

## When to suppress

- The call is in non-hot code where readability is preferred

## Minimum TFM

Any
