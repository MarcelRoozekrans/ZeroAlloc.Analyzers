# ZA1102: Dispose CancellationTokenSource

## Description

Detects `new CancellationTokenSource()` expressions that are not wrapped in a `using` statement or `using` declaration. `CancellationTokenSource` holds unmanaged resources (a `WaitHandle` and timer callbacks) that should be disposed to avoid resource leaks.

## Violation

```csharp
void M()
{
    var cts = new CancellationTokenSource();
    cts.Token.ThrowIfCancellationRequested();
}
```

## Fix

```csharp
void M()
{
    using var cts = new CancellationTokenSource();
    cts.Token.ThrowIfCancellationRequested();
}
```

## When to suppress

- The `CancellationTokenSource` is stored as a field and its lifetime is managed by the containing class (e.g., disposed in `Dispose()`)
- The `CancellationTokenSource` is passed to a long-lived object that manages its disposal
