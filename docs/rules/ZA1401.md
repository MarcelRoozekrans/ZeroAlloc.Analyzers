# ZA1401: Use static lambda to prevent accidental closures

## Description

Detects lambda expressions that do not capture any variables from the enclosing scope and are not already marked `static`. Adding the `static` modifier prevents accidental future captures and avoids unnecessary closure allocations.

## Violation

```csharp
var items = new[] { 1, 2, 3 };
var r = items.Where(x => x > 0);
```

## Fix

```csharp
var items = new[] { 1, 2, 3 };
var r = items.Where(static x => x > 0);
```

## When to suppress

- The lambda is expected to capture variables in the near future
- Readability is preferred over the minor performance benefit

## Minimum TFM

net5.0
