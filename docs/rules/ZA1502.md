# ZA1502: Avoid finalizers

## Description

Detects classes that define a finalizer (destructor). Objects with finalizers are promoted to Gen1+ by the garbage collector, adding significant overhead. The GC must place finalizable objects on the finalization queue and process them in a separate pass, which delays collection and increases memory pressure.

## Violation

```csharp
class MyResource
{
    ~MyResource() { /* cleanup */ }
}
```

## Fix

```csharp
class MyResource : IDisposable
{
    public void Dispose()
    {
        // Perform cleanup here
    }
}
```

## When to suppress

- The class wraps a native resource with no other safe handle or disposable wrapper available
- The finalizer acts as a safety net for a critical unmanaged resource that must never leak
